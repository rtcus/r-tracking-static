# R跟单台系统 - 静态资源仓库

这个仓库托管 R跟单台系统 所需的所有第三方静态资源，用于加速页面加载。

## 使用方式

在 HTML 文件中使用以下链接：

### CSS 资源
```html
<!-- Bootstrap 5.2.3 -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/YOUR_USERNAME/r-tracking-static@main/css/bootstrap.min.css">

<!-- Flatpickr 4.6.13 -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/YOUR_USERNAME/r-tracking-static@main/css/flatpickr.min.css">

<!-- Font Awesome 6.0.0 -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/YOUR_USERNAME/r-tracking-static@main/css/font-awesome.min.css">
